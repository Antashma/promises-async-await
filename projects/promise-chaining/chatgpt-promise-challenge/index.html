Practice Problem: Fetch User Data and Posts

Objective

Create a function that fetches user data and their posts from a mock API. The function should use Promises to handle the asynchronous operations.

Instructions

  1. Mock API Endpoints:

    • `https://jsonplaceholder.typicode.com/users/1`: This endpoint returns the data of a user with ID 1.
    • `https://jsonplaceholder.typicode.com/posts?userId=1`: This endpoint returns the posts created by the user with ID 1.

  2. Create a function `fetchUserDataAndPosts`:

    • This function should fetch the user data from the first endpoint.
    • After successfully fetching the user data, it should then fetch the user's posts from the second endpoint.
    • The function should return a Promise that resolves to an object containing both the user data and their posts.

  3. Error Handling

    Ensure that your function properly handles any errors that might occur during the fetch operations.

  4. Output Example:

    The resolved value of the Promise should be in the following format:

    javascript

    
                    {
                        user: { /* user data */ },
                        posts: [ /* array of posts */ ]
                    }
