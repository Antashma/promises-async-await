Practice Problem #2: Fetch & Process Data

Create a program that simulates a sequence of asynchronous operations using promises. Each operation should consist of fetching some data, processing the data, and then logging the results. Ensure that all operations are logged in the correct order using microtasks.

  1. Implement three asynchronous functions:
    • fetchData(url): Simulates fetching data from a URL. It should return a promise that resolves with the data after a random delay (between 100 and 500 milliseconds).
    • processData(data): Simulates processing the fetched data. It should return a promise that resolves with the processed data after a random delay (between 100 and 300 milliseconds).
    • logResults(data): Logs the processed data to the console.
  2. Chain these functions together to simulate a sequence of operations:
    • Fetch data from three different URLs.
    • Process the fetched data.
    • Log the processed data to the console.
  3. Ensure that the operations for each URL are logged in the correct order using microtasks (promises).

Example Usage

javascript


    const urls = [
        'https://api.example.com/data1',
        'https://api.example.com/data2',
        'https://api.example.com/data3'
    ];

    performOperations(urls);
